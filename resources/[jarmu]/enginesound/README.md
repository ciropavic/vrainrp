# enginesound
Change Vehicle Engine Sound and Sync to all server One Sync/One Sync Infinity (included carsound pack from gta5mods)

- Installation:
- Drag enginesound to your resource folder and start at server.cfg
- start enginesound
- Usage: 
- /changesound k20a
- /changesound elegy
- /changesound toysupmk4
- /changesound blista
